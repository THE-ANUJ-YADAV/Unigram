# Unigram - Community App

A premium community application built with React + Vite.

## Features
- **Dashboard**: Overview of activities.
- **Lost & Found**: Report and find lost items.
- **Resources**: Access PYQs and Notes.
- **Announcements**: Latest news and events.
- **Marketplace**: Buy and sell items.
- **Helping Hands**: Community support.

## Setup Instructions

Since Node.js was not detected in the environment, you need to install it first.

1. **Install Node.js**: Download and install from [nodejs.org](https://nodejs.org/).
2. **Install Dependencies**:
   Open a terminal in this folder and run:
   ```bash
   npm install
   ```
3. **Run the App**:
   ```bash
   npm run dev
   ```
4. **Open in Browser**:
   Click the link shown in the terminal (usually `http://localhost:5173`).

## Tech Stack
- React
- Vite
- React Router DOM
- Lucide React (Icons)
- Vanilla CSS (Glassmorphism Design)
